Panggilan untuk Berkontribusi!

Website Karang Taruna Banjarsari adalah ruang digital milik kita bersama. Kami percaya, ide terbaik datang dari banyak kepala.

Apakah Anda mau ikut serta dalam pengembangannya? Anda tidak perlu bisa coding! Banyak cara untuk berkontribusi: menjadi penulis (copywriter) untuk artikel, menambahkan dokumentasi foto/video, atau bahkan sekadar menyumbang ide dan tweet lucu untuk ditampilkan.

Jika "Ya", Anda akan diarahkan ke admin web. Anda bisa mengirimkan materi kontribusi dalam format apa pun (ide, tulisan, foto, video, dll). Setiap bantuan, sekecil apapun, sangat berarti!
